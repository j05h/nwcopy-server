Getting started

Here’s how to get rolling:

  1. Log in with your github.com account

    Pretty sure you don't want yet another account!.

  2. Install the nwcopy gem

    gem install nwcopy

  3. Configure it with your access key

    echo 'export NWCOPY_KEY=foo' >> ~/.profile

  4. Get to copy/pasting

    nwcopy locally; nwpaste remotely
